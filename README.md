WordPressi teema

Projekti eesmärk on teha õppimise otstarbel lihtne WordPressi teema kodulehe majutamiseks
Esilehel on päis, lühikesed kokkuvõtted artiklitest ning lingid artiklite juurde.

Selleks, et projekti kasutada, tuleb see lisada WordPressi teemade kataloogi, seadistada menüü.

Teema on alles arendusjärgus ning puudub viis pildi näitamiseks artikli juures
